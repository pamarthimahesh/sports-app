# Sports Playground Smart Booking System

## Overview
Sports Playground Smart Booking System is a web-based application that allows users to search, book, and manage sports playground reservations online.

## Features
- User Registration and Login
- Playground Search and Booking
- Booking Management
- Schedule Management
- CRUD Operations
- Responsive User Interface

## Technologies Used
- Java
- MySQL
- JDBC
- HTML
- CSS
- JavaScript
- REST APIs
- Postman

## Responsibilities
- Developed frontend and backend modules.
- Implemented booking and scheduling functionality.
- Integrated database operations using MySQL and JDBC.
- Tested APIs using Postman.
- Collaborated with team members to improve application performance.
